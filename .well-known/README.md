# .well-known/

`assetlinks.json` lands here — it's how the Android TWA app proves it owns this
domain, which removes the browser URL bar from the app. It is GENERATED by
`android-twa/build-android.sh` (it needs the signing key's SHA256 fingerprint).

After the first Android build:
  cp android-twa/assetlinks.json .well-known/assetlinks.json
  git add .well-known/assetlinks.json && commit && push

Until then this file is a placeholder so the directory exists in the deploy.
