## Fast capture tips
- **Desktop 1280×800:** Chrome → DevTools → device toolbar → Responsive → set 1280×800 → screenshot via DevTools (⋮ → Capture screenshot).
- **Phone 9:16:** DevTools → iPhone 12 Pro → ⋮ → Capture screenshot. Gives a clean 390×844 (scale up to 1080×2340 if Play wants larger).
- Sign in first (use the ✦ Begin pill) so signed-in surfaces (lineage, kept moments) have real data.
- The gallery already has real public cards — no setup needed.
- For the feature graphic / promo tile, the SVGs in `branding/vintinuum/social/`
  are good raw material.

## Where to put them
- Chrome: upload directly in the Web Store Developer Console listing.
- Play: upload in Play Console → Store listing → Graphics.
- (Optional) keep masters in `branding/vintinuum/store/` for reuse.
