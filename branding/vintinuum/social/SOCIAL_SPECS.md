## Recommended Usage

- YouTube profile: export `profile-square-1080.svg` to PNG and center-crop if needed
- YouTube banner: use `youtube-banner-2560x1440.svg`
- X profile: export `profile-square-1080.svg`
- X header: use `x-header-1500x500.svg`
- Facebook page profile: export `profile-square-1080.svg`
- Facebook cover: use `facebook-cover-1640x624.svg`
- LinkedIn profile or company avatar: export `profile-square-1080.svg`
- LinkedIn company cover: use `linkedin-company-cover-1128x191.svg`
- Instagram profile: export `profile-square-1080.svg`
- Instagram story / reel cover: use `instagram-story-1080x1920.svg`
- TikTok profile: export `profile-square-1080.svg`
- TikTok vertical intro / story cover: use `tiktok-vertical-cover-1080x1920.svg`
- Discord server icon: use `discord-server-icon-1024.svg`
- Discord server banner: use `discord-server-banner-960x540.svg`
- Twitch profile image: export `profile-square-1080.svg`
- Twitch profile banner: use `twitch-profile-banner-1200x480.svg`

## Dimension Notes

Official dimensions confirmed from platform help where available:

- YouTube banner: `2560 x 1440`, safe area `1235 x 338`
  Source: https://support.google.com/youtube/answer/10456525
- LinkedIn company cover: `1128 x 191`
  Source: https://www.linkedin.com/help/linkedin/answer/a566603

For X, Facebook, Instagram, and TikTok, this pack uses current platform-safe working sizes intended to preserve cropping flexibility and maintain sharpness across devices. Profile images are supplied as a universal `1080 x 1080` master because those platforms render avatars responsively.

## Production Notes

- Keep the logo mark centered in all avatar exports
- Avoid pushing the wordmark to the extreme left or right in banner exports
- On YouTube, keep key text inside the dashed safe-area frame in the SVG
- For TikTok and Instagram, the vertical masters are intentionally clean enough to work as both cover art and short-form end cards
